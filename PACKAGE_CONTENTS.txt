DEPLOYMENT STEPS
----------------
1. Extract tob_calculator.zip to desired location
2. Double-click START_TOB_CALCULATOR.bat
3. Wait for dependency installation (first run only)
4. Browser opens automatically at http://localhost:5000
5. Upload broker PDFs and calculate TOB

CALCULATION METHODOLOGY
-----------------------
- TOB Rate: 0.35% (0.0035)
- Max per transaction: €1,600
- Exchange rates: ECB official XML feed
- Grouping: Same stock + same day + same type = 1 transaction
- Day trades: Both buy and sell sides taxed separately

OUTPUT FILES
------------
Each calculation generates 4 files:
1. belgian_tob_tax_TIMESTAMP.xlsx - Excel report
2. belgian_tob_tax_TIMESTAMP.csv  - Belgian CSV format
3. belgian_tob_tax_TIMESTAMP.pdf  - PDF report
4. TOB_Tax_Summary_TIMESTAMP.md   - Markdown summary
